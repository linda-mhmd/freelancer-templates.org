# freelancer-templates.org — Freelancer Template Library

**Domain:** freelancer-templates.org

## Purpose
Main site for a freelancer template library offering professional templates for contracts, proposals, invoices, project plans, and other freelance business documents.

## Key Features / Sections
- Template catalog by category
- Preview & download system
- Free & premium template tiers
- Customization guides
- Industry-specific template packs
- User submissions & community templates
